# Design System

## Colors

| Token | Usage |
|-------|-------|
| `bg-green-500` | Enabled / active states |
| `bg-red-500` | Disabled / error states |
| `bg-yellow-500` | Warning / archived states |
| `bg-blue-500` | Information / links |
| `bg-gray-100` | Backgrounds |
| `bg-gray-800` | Text |

## Environment Colors

Each environment gets a distinct color for visual differentiation:
- Development: `blue`
- Staging: `yellow`
- Production: `red`

## Component Patterns

- Cards for flag list items
- Badges for status and environment indicators
- Tabs for environment switching on flag detail
- Modals for create/edit flows
- Confirmation dialogs for destructive actions
